    Código JavaScript

    JavaScript tiene sus propias reglas para la sintaxis, aunque respeta los estándares de muchos lenguajes de programación lógicos. Nuestro código JavaScript se asocia al archivo HTML que se carga en el navegador. Tenemos dos maneras de escribir código JavaScript en nuestras aplicaciones web.

    ¿Cómo escribir código JS?

    En un archivo individual con extensión .js Ejemplo: mi-archivo.js Se vincula con la etiqueta script y el atributo “src” Recuerda no utilizar espacios ni mayúsculas en los nombres de archivo.
